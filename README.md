# Arduino Light Sensor LED

## Description
This project uses an Arduino and a light sensor to control an LED.

## How it works
- When the light sensor is covered by hand, the LED turns on.
- When there is enough light, the LED turns off.

## Components
- Arduino Uno
- Light sensor
- LED
- Resistor
